## 🚀 Featured Projects

### 🌍 VistaVoyage — AI Travel Discovery App
> **Stack:** Flutter 3.x · Dart · Supabase · PostgreSQL · Groq API (Llama 3.3 70B) · Git

A cross-platform mobile app (iOS & Android) where users explore 10 curated global destinations and get **personalised AI travel itineraries generated in real time** through a conversational chat interface powered by Groq's Llama 3.3 70B model.

**What makes it interesting:**
- 🤖 Real-time AI itinerary generation with low-latency Groq API responses
- ☁️ Supabase auth + PostgreSQL cloud backend with live data sync
- 💾 Persistent user favourites and session management
- ✨ Polished animated UI — typing indicators, swipe-to-delete, pull-to-refresh, live search

🔗 [View Repository](https://github.com/QaswarSarfrazcodes/VistaVoyage-Semester-Project)
